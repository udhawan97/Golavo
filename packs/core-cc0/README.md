# core-cc0 pack

The primary Golavo data pack. **CC0 / public domain only.**

- openfootball — club fixtures/results
- martj42/international_results — internationals: results, scorers, shootouts
- Wikidata (structured data) — entity resolution & historical facts

Redistributable with no attribution required (credited voluntarily in `NOTICE`).

**Do not** add any ODbL, proprietary, or BYOK source to this pack. CI
(`check_license_isolation.sh`) fails the build if OpenLigaDB/ODbL is referenced
here.
