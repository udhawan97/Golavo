# Phase 6 correction-proposal acceptance

Phase 6 is acceptable only when tests prove that local correction activity leaves
the match index, sealed artifacts, settlement, calibration, model inputs and all
redistributable packs byte-identical. The installed app must work offline, retain
drafts across restart, render hostile evidence as plain text, block ODbL export,
and perform no network action until the user explicitly opens a contribution page.

`accepted_local` means a local annotation. `submitted` is only the user's local
attestation that they completed an external workflow; Golavo does not verify it.
