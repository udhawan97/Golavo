# golavo-ui

The Golavo interface: React + TypeScript + Vite. **AGPL-3.0.**

```bash
npm install
npm run dev      # http://127.0.0.1:5173
npm run build
```

The UI never talks to data sources directly — it calls the local `golavo-server`.
The full interface (Matchday, Fixture Room, Forecast Theatre, After the Whistle)
lands in Phase 2. This scaffold is a placeholder shell.
