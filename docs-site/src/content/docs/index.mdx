      <p class="gv-kicker">Who gets to touch the numbers?</p>
      <h2 id="gv-ai-title" class="gv-chapter-title">The engine decides.<br/><em>AI may explain.</em></h2>
      <div class="gv-body">

        bundle. Turn every AI setting off and the probabilities, score matrix,
        seal, and calibration remain byte-for-byte the same.
      </div>
      <ul class="gv-check-list">

        <li>Every narrative claim cites allowed evidence</li>
        <li>Any guard failure returns to local-only</li>

    <div class="gv-chapter-copy gv-chapter-copy--wide">
      <p class="gv-kicker">Private by architecture, not by promise</p>
      <h2 id="gv-architecture-title" class="gv-chapter-title">A local app with<br/><em>one numeric authority.</em></h2>
      <div class="gv-body">

        validated read-only views. FastAPI serves contracts. The deterministic
        Python core remains the only place a forecast can be made.
