  <h2 class="gv-section-title">The engine owns the numbers. AI only ever cites.</h2>
  <p class="gv-section-lead">
    Two layers, one hard rule between them. The statistics own every probability.
    The AI can research and explain — but it can never silently change a number.
    That separation is structural, not a promise.
  </p>

  <div class="gv-contract">
    <div class="gv-pillar gv-pillar--engine">
      <span class="tag">Statistical engine</span>
      <h3>Owns the probabilities</h3>
      <ul>
        <li>Every probability, score matrix, and count distribution</li>
        <li>Reruns only when a <em>confirmed</em> fact becomes a typed feature</li>
        <li>Forward-backtested; nothing ships unless it beats the baselines</li>
      </ul>
    </div>
    <div class="gv-pillar gv-pillar--ai">
      <span class="tag">AI layer · optional</span>
      <h3>Cites, explains, researches</h3>
      <ul>
        <li>Narrative, scenario explanation, sourced research</li>
        <li>Proposes typed features for the engine to consider</li>
        <li>May <em>never</em> state a number not in its evidence bundle</li>
      </ul>
    </div>
  </div>
