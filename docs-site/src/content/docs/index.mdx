  <section class="gv-substrate" aria-labelledby="gv-substrate-title">
    <p class="gv-substrate-head" id="gv-substrate-title">Underneath every stage · always true, never a step</p>

    <div class="gv-stage-grid">
      <div>
        <h3>The engine decides. AI may explain.</h3>
        <p>
          Deterministic code produces the probabilities and score matrix. The
          Commentator’s Notebook turns fixed, source-backed facts into a glanceable
          briefing with detail behind disclosure. Optional local or BYOK AI receives
          a guarded read-only bundle and can disappear without changing the match
          analysis by a single digit.
        </p>
        <ul class="gv-check-list">
          <li>Every displayed number is engine-owned</li>
          <li>Model voices stay separate instead of being averaged</li>
          <li>Research fetches only a source you selected and never makes it authoritative</li>
          <li>Unsupported AI output fails back to deterministic analysis only</li>
        </ul>
        <a class="gv-text-link" href="/Golavo/ai/providers/#set-up-ollama-inside-golavo">Set up local AI <span aria-hidden="true">↗</span></a>

    <div class="gv-substrate-grid">
      <div>
        <h3>Local data, with receipts</h3>
        <p>
          Refresh approved CC0 sources manually or with consent. Followed matches get
          targeted checks and deduplicated change history. OpenLigaDB stays an
          optional, display-only ODbL overlay whose bytes never join CC0 or model data.
        </p>
        <a class="gv-text-link" href="/Golavo/data/sources/">See the sources <span aria-hidden="true">↗</span></a>
      </div>
      <div>
        <h3>Leagues, Europe, and eras</h3>
        <p>
          Follow competition-local strength and workload, inspect a domestic season’s
          fixture gate, or switch between separate World Cup model voices. Historical
          research stays collapsed behind its exact competition and era, and is never
          a model input.
        </p>
        <a class="gv-text-link" href="/Golavo/competition-analytics/">See the competitions <span aria-hidden="true">↗</span></a>
      </div>
      <div>
        <h3>Private by architecture</h3>
        <p>
          Tauri supervises a token-protected loopback service. React presents validated
          views. FastAPI serves typed contracts. The deterministic Python core is the
          only place a forecast can be made; optional narration stays outside that
          write path.
        </p>
        <a class="gv-text-link" href="/Golavo/architecture/">Inspect the architecture <span aria-hidden="true">↗</span></a>
