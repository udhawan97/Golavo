---
title: Casual vs Expert
description: Two presentation modes over the same numbers. Different depth never means different certainty.
---

Golavo presents the **same** forecast at two depths. The numbers do not change between modes — only how much detail is shown.

| | Casual | Expert |
|---|---|---|
| Headline | verdict bar + plain-language summary | verdict bar with explicit intervals |
| Facts | top 3 cited facts | full Commentator's Notebook |
| Score matrix | hidden by default | full matrix with tail bucket |
| Features | hidden | typed-feature table with effect sizes |
| Versions | hidden | model/feature versions, source hashes |

Two rules hold in both modes:

1. **Presentation never changes displayed certainty.** A prettier layout does not imply a more confident forecast.
2. **AI text is always visually subordinate to the sealed numbers.** The engine speaks first.

Casual-mode phrasing is generated from the same distribution via fixed phrase bands (e.g. a 40–45% favorite reads "slightly favored"), not by AI.
